使用 DSpinBox::defaultValue 获取默认值，使用 DSpinBox::setDefaultValue 设置默认值, 点击控件上的 reset 按钮会设置为此值。

bool DSpinBox::isAlert ( ) const

表示当前控件是否处于警告状态的属性.

使用 DSpinBox::isAlert 获取属性当前状态，使用 DSpinBox::setAlert 设置属性的状态。

返回
当前状态处于警告状态，返回 true，否则返回 false.

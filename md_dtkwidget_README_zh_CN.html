Deepin Tool Kit Widget

Deepint Tool Kit Core(DtkWidget) 提供各种UOS风格dtk基础控件.

您应该首先阅读 Deepin应用程序规范.

依赖

编译依赖

  • Qt >= 5.6

安装

从源代码构建

  1. 确保已经安装了所有的编译依赖.
    $ sudo apt build-dep ./
    如果需要使用 qtcreator 的设计功能，可以 ：
    $ sudo apt install qttools5-dev
    # build
    $ cmake ./plugin/dtkuiplugin -B build -DINSTALL_PLUGIN=ON -DCMAKE_INSTALL_PREFIX=/usr
    $ cmake --build build -j$(nproc)
    # install
    $ cd build
    $ sudo make install
  2. 构建 dtkwidget :
$ cmake -B build
$ cmake --build build -j$(nproc)
  1. 安装:
$ cd build
$ sudo make install

使用dtkwidget的项目

下面是使用dtkwidget的开源项目:(如果您想添加属于自己的开源项目请给我们提交PR)

  • DMarked: 使用Dtk构建的Markdown编辑器
  • DtkTimer: 使用DtkWidget构建的时钟
  • simple-image-filter: 使用DtkWidget构建的图像处理软件
  • SparkStore: 星火商店
  • WingHexExplorer: 使用DtkWidget构建的强大的十六进制编辑器
  • WingTool: 使用DtkWidget构建的基于插件的工具箱
  • ScreenLight: 使用DtkWidget构建的用于调节屏幕亮度的小工具

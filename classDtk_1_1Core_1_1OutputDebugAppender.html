Writes the log record to the windows debug log. \reimp

Writes the log record to the windows debug log.

The time parameter indicates the time stamp. The level parameter describes the LogLevel. The file parameter is the current file name. The line parameter indicates the number of lines to output. The func parameter indicates the function name to output. The category parameter indicates the log category. The msg parameter indicates the output message.
